Resolução Analítica Detalhada

Questão 1: Taxonomia Geográfica e Evolução Tecnológica

Gabarito Correto: D

1.1. LAN (Local Area Network)

A afirmação I define corretamente a LAN. Uma LAN não é definida apenas por metros, mas pelo controle administrativo e características de sinal (baixa latência, alta velocidade). A menção ao broadcast é crucial, pois protocolos como Ethernet baseiam-se na difusão para descoberta de serviços.

1.2. WMAN e WiMAX

A afirmação II aborda a WMAN. Enquanto o Wi-Fi (802.11) é para ambientes internos (LAN), o WiMAX (802.16) foi projetado para a "Última Milha" metropolitana, operando com potências maiores e frequências licenciadas.

1.3. WAN: Frame Relay vs MPLS

A afirmação III está correta. O MPLS (Multiprotocol Label Switching) substituiu o Frame Relay por permitir engenharia de tráfego e QoS robusto. Diferente do roteamento IP tradicional que analisa todo o cabeçalho, o MPLS comuta baseado em "labels" curtos, agilizando o backbone.

Questão 2: Topologias Físicas e Centralização

Gabarito Correto: C

A questão exige alta disponibilidade e fácil manutenção. A topologia em Estrela é a norma moderna.

Diferente do barramento (onde um cabo quebrado derruba a rede), na estrela, a falha de um cabo afeta apenas um PC. O uso de um Switch Gerenciável (ao invés de Hub) é vital, pois cria microsegmentação, evitando colisões e permitindo controle de tráfego, essencial para Data Centers.

Questão 3: O Papel do Roteador no Modelo OSI

Gabarito Correto: B

O Roteador opera na Camada de Rede (L3). Sua função é interconectar redes lógicas diferentes usando endereços IP.

A característica chave cobrada é o Controle de Broadcast. Roteadores não propagam pacotes de broadcast (255.255.255.255) de uma interface para outra, delimitando o domínio de broadcast e impedindo que o tráfego de descoberta de uma rede local inunde a WAN.

Questão 4: Dualidade Topológica (Física vs. Lógica)

Gabarito Correto: B

Fisicamente, vemos uma estrela (cabos indo para um centro). Porém, se o centro é um Hub, eletricamente todos os pinos estão conectados. O sinal de um PC vai para todos os outros simultaneamente.

Isso cria uma Topologia Lógica em Barramento. Como o meio é compartilhado, colisões podem ocorrer, exigindo o uso obrigatório do protocolo CSMA/CD (Carrier Sense Multiple Access with Collision Detection) para gerenciar o acesso.

Questão 5: A Física dos Meios de Transmissão

Gabarito Correto: D
  • I (Falso): Fibra óptica é um meio guiado (luz presa no núcleo). Meios não guiados são wireless.
  • II (Verdadeiro): Cabos STP (Shielded Twisted Pair) possuem blindagem metálica que age como Gaiola de Faraday, essencial para ambientes industriais com alta interferência (EMI).
  • III (Verdadeiro): A fibra monomodo tem núcleo estreito (~9 micrômetros), permitindo que a luz viaje em um único modo axial. Isso elimina a dispersão modal, permitindo alcances de dezenas de quilômetros (WANs).

Questão 6: Matemática das Topologias (Malha)

Gabarito Correto: D

Para uma Malha Total (Full Mesh), a fórmula é N * (N-1) / 2.

Com N = 6 roteadores:

6 * (5) / 2 = 30 / 2 = 15 links.

Isso demonstra o alto custo de cabeamento da malha total.

Questão 7: Comutação e Domínios de Broadcast

Gabarito Correto: B

Switch (L2): Ao receber um broadcast (FF:FF:FF:FF:FF:FF), ele deve propagá-lo para todas as portas exceto a de origem. Isso estende o domínio de broadcast.

Roteador (L3): Ao receber um broadcast, ele o descarta (não roteia). Isso delimita o domínio de broadcast, impedindo tempestades de pacotes entre sub-redes.

Questão 8: A Realidade das Redes Wi-Fi (WLAN)

Gabarito Correto: C

O Wi-Fi é Half-Duplex (o rádio não transmite e recebe ao mesmo tempo na mesma frequência). Como não é possível detectar colisão "ouvindo" o meio enquanto se fala (como no cabo), usa-se o CSMA/CA (Collision Avoidance).

A topologia lógica é de Barramento Compartilhado, pois todos competem pelo mesmo meio aéreo (espectro de frequência).

Questão 9: Arquiteturas Administrativas

Gabarito Correto: B

O cenário descreve uma rede sem servidor central (controlador de domínio). Cada máquina gerencia seus próprios recursos e usuários.

Isso é o modelo Peer-to-Peer (Ponto-a-Ponto). A principal limitação é a descentralização da segurança e gerenciamento: senhas não são sincronizadas, backup é individual e a administração se torna caótica conforme o número de usuários cresce.

Questão 10: Padrões de Cabeamento EIA/TIA

Gabarito Correto: C

Para Gigabit Ethernet (1000Base-T), o padrão exige cabos que suportem frequências de 100MHz e tenham parâmetros de diafonia controlados.

O Cat 5e (Enhanced) é a categoria mínima para garantir 1 Gbps estavelmente. O conector padrão para par trançado em redes de dados é o RJ-45 (8P8C).
